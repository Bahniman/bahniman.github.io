🛰️ Settlement Rail

Realium

A settlement rail for public works. Turns physical, site-verified construction progress (e-MB logs) into a bank-grade, financeable receivable, reducing contractor dues wait times from 148 days to 1 day. Consolidates both Groundtruth & Surety engines.

  • Evidence Layer: Geo-tagged photo attestation & Ed25519 signatures.
  • Authority Layer: Signed, revocable, state-scoped certifier mandates.
  • Liquidity Layer: Automated banking waterfall & reliability flywheels.

🏛️ Secure Vault

Heirloom

Organizational memory as an asset the firm owns. Scopes database queries through role-based cryptography, ensuring context is model-independent and survives vendor switches or partner departures.

  • Access Guard: Pre-retrieval scope filtering based on user credentials.
  • Provenance Chain: Hard-coded audit citations linked to document nodes.
  • Open Portability: Vendor-free export schemas in neutral JSON structures.

🚪 Store Analytics

Turnstile

Commerce analytics and interface negotiation for the AI agent economy. Classifies user sessions (human vs AI agents), recovers falsely declined bot checkouts, and serves machine-readable API storefronts.

  • Gate Classifier: Behavioral signal processing (mouse traces, load cadences).
  • Attribution Flow: Accurate routing of automated delegated purchases.

Windtunnel

A Monte Carlo simulator for testing strategic price and plan changes. Generates synthetic customer populations modeled from real support histories, reviews, and transaction counts to estimate risks.

  • Synthetic Cohorts: Friction sensitivity fitted to customer review text.
