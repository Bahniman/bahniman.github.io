Turnstile

How much of your online store's traffic is actually AI, not people?

What is this, in plain English? More and more online shopping is done by AI assistants on people's behalf — they browse, compare, and buy. But every store's analytics assumes visitors are humans. So stores can't answer a basic question: how many of my visitors are software? Turnstile spots the AI visitors, shows the store the real breakdown, and then serves those AI visitors a clean price list instead of a human web page. Press "Run a day of traffic" to see it work.
Jargon decoder (tap to open)
TermWhat it actually means
Agent trafficWebsite visits made by AI software, not a human clicking around.
ClassifierThe logic that decides "human or AI?" from clues in how the visit behaves.
SignalsThe clues: does the mouse move? did it load images? how fast did it act? does it announce itself?
AttributionCorrectly crediting a sale to where it came from — e.g. "this purchase came via an AI assistant."
AEO (Agent Experience Optimization)Like SEO was for Google, but for AI shoppers: giving agents a clean, machine-readable offer instead of a human web page.
Offer feedA simple price+stock list built for machines to read, at a fixed address on the site.

The clever part: sell to the agents (AEO)

Instead of showing an AI a carousel and a cookie banner, Turnstile serves it this, at /.well-known/agent-offers.json:
{ "store": "Demo D2C Store", "currency": "INR", "products": [ { "name": "Insulated bottle 750ml", "price": 1499, "in_stock": true, "ships_in_days": 3, "buy_endpoint": "/api/checkout?sku=BTL-01" }, { "name": "Insulated bottle 1L", "price": 2199, "in_stock": true, "ships_in_days": 5, "buy_endpoint": "/api/checkout?sku=BTL-02" } ] }
The agent gets clean data (fewer wrong orders); the store gets a correctly-attributed sale. Both sides win — which is why the standard would stick.

Why this matters

Search traffic built an $80 billion optimization industry (SEO). Agent traffic is at the same starting line today — measurable, growing, and unmeasured by anyone. Whoever builds the tool that measures it gets to define the category, and the more stores join, the better the detection gets — a data advantage a latecomer can't simply copy.
